# Superhero Showdown

Superhero Showdown is a fully-responsive, single-page app consuming JSON data from a RESTful Superhero API available [here] (http://superheroapi.com/). 

The API provides not only superhero and supervillan biographical information but also video game-like stats. 
This realization motivated me to build an original _"Battle"_ mechanic for the characters.  

>The project was created with vanilla Javascript, CSS and HTML. 
It is deployed to Heroku, which required a few lines of node.js + Express. The Procfile is Heroku specific. 

[See the live demo here] (https://superhero-showdown-app.herokuapp.com/)
