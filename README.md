# Superhero Showdown

"Superhero Showdown" is a fully-responsive, single-page app consuming JSON data from a superhero API. 

The API provides superhero and supervillan biographical information and also video game-like stats. 
The characters' stat data inspired me to build an original _"Battle"_ mechanic for the characters as an interactive web app.  

>The project was created with vanilla Javascript, CSS and HTML. 
It is deployed to Heroku, which required a few lines of Express. The Procfile is Heroku-specific. 

[See the live demo here](https://superhero-showdown-app.herokuapp.com/)
